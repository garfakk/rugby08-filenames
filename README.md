This project contains the reversed filenames of the Rugby 08's `data.gob` archive.
The `data.gob` stores the MD5 hash of the filepaths (without extension) instead of the actual path.

The cracked hashed are in `data_gob.csv` of this repository.
So far 5444/5445 hashes have been reversed, only 96fc228cf8964b9d5f4a0f63d190968f.txt remains uncracked.

Any contribution is welcomed to find the last filepath!
